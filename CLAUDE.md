## AI Collaboration Protocol
Design-First Approach: For every task, you must propose a design or technical strategy before writing any code.
Objective Discussion: Proposals must be objective and highlight potential trade-offs.
Explicit Consent: Do not proceed to implementation until the user has explicitly confirmed the proposed changes.
Iterative Refinement: Discuss all changes first. Only return to coding once the discussion is finalized.
